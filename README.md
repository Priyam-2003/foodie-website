# 🍔 Foodie – Food Ordering Website

A responsive and interactive **Food Ordering Website** built using **HTML, CSS, JavaScript**, and **JSON**.  
It allows users to browse a menu, add items to the cart, adjust quantities, and view the total price.  
The site also features a hero section, services, customer reviews, app promotion, and newsletter subscription.

---

## 🚀 Features
- 📱 **Responsive Design** (works on desktop & mobile with a hamburger menu)
- 🛒 **Shopping Cart Functionality**
  - Add items to cart
  - Increase / decrease item quantity
  - Auto update total price and cart count
  - Remove item when quantity is 0
- 🎨 **Modern UI** with custom CSS and Font Awesome icons
- 🔄 **Swiper Slider** for customer reviews
- 📂 **Dynamic Product List** loaded from `product.json`

---

## 🛠️ Tech Stack
- **HTML5** – Structure
- **CSS3** – Styling (Flexbox, Responsive Design)
- **JavaScript (Vanilla)** – Functionality (cart, menu toggle, slider)
- **JSON** – Product data
- **Swiper.js** – Review slider
- **Font Awesome** – Icons

---

## 📂 Project Structure
```
 ├── index.html Main HTML file
 ├── style.css # Stylesheet
 ├── main.js # JavaScript (cart, menu, swiper, etc.)
 ├── product.json # Product data (id, name, price, image)
 └── imgs/ # Images (products, profile, app, etc.)
 ```
